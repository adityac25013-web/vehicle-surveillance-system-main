Surveillance Enhanced Vehicle Tracking System
=============================================

This project implements a real-time vehicle surveillance and tracking system using:

- Python, OpenCV, YOLOv8, DeepSORT
- Tesseract / EasyOCR for license plate OCR
- SQLite / MySQL for structured storage
- Flask as the backend API
- HTML, CSS, JavaScript for the frontend dashboard

Project layout:

- `backend/` – Flask API, detection pipeline, database layer
- `frontend/` – HTML/CSS/JS dashboard consuming the backend APIs/WebSockets

You can open this folder in Cursor and start working on the backend and frontend separately.
